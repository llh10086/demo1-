# 用户查询功能说明

## 功能概述

实现了用户查询功能，张三账号登录后可查看所有用户信息。

## 使用说明

### 1. 启动应用

```bash
./gradlew bootRun
```

### 2. 访问登录页面

打开浏览器访问：http://localhost:8080/login

### 3. 登录账号

使用以下账号登录：
- **用户名**: 张三
- **密码**: 123456

### 4. 查看用户列表

登录后有两种方式查看用户列表：
1. 在首页点击"👥 查看所有用户"按钮
2. 直接访问：http://localhost:8080/users

### 5. 测试用户

系统会自动创建以下测试用户：
- 张三 (USER) - zhangsan@example.com
- 李四 (USER) - lisi@example.com  
- 王五 (ADMIN) - wangwu@example.com

## 实现细节

### 后端实现

1. **UserService** - 添加了 `getAllUsers()` 方法获取所有用户
2. **AuthController** - 添加了 `/users` 接口返回用户列表页面
3. **数据初始化** - 使用 `@PostConstruct` 自动创建测试用户

### 前端实现

1. **users.html** - 用户列表页面，展示所有用户信息
2. **home.html** - 添加了查看用户列表的链接

### 安全配置

- 所有用户相关页面都需要登录后才能访问
- 使用 Spring Security 进行身份验证

## 技术栈

- Spring Boot 4.0.6
- Spring Security
- Spring Data JPA
- Thymeleaf
- MySQL
